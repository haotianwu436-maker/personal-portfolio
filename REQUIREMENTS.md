# 项目需求文档

## 📋 项目概述

**项目名称：** Personal Portfolio Website (个人网站)

**项目目标：** 创建一个极简、有温度、人文气质的个人网站，用于长期展示理念、项目与社区实践。

**核心理念：** 
> 半梦半醒，永远年轻，永远热泪盈眶。做一个有温度、有热情的人。

**网站定位：** 不是"冷冰冰的技术简历"，而是有情绪、有生命感的个人主页。

## 🎯 核心需求

### 1. 整体定位与气质

#### 气质关键词
- 克制：不过度装饰，让内容说话
- 真诚：真实表达个人想法和经历
- 有力量但不张扬：专业但不傲慢
- 人文感：关注人与人的连接
- 年轻但不浮躁：有思考深度

#### 设计参考
- 参考网站：https://brittanychiang.com
- 风格：极简、留白、纵向单页、排版重要

### 2. 设计风格要求

#### 极简高级 (Minimal & Editorial)
- 大量留白，内容不拥挤
- 编辑感强，像阅读高端文学杂志

#### 颜色方案
- **主色**：黑 / 深灰 / 米白
- **背景**：温暖米白 (#F5F1ED)
- **文字**：深炭灰 (#3C3C3C)
- **强调色**：森林绿 (#2D5A4F) - 象征生长与社区

#### 字体
- **Display Font**：Playfair Display（优雅、编辑感）
- **Body Font**：Inter（现代、可读性强）

#### 动效
- 非炫技，只在 hover / 滚动时轻微淡入、上移、阴影变化
- 移动端优先，手机打开必须舒服

### 3. 页面结构

#### 1️⃣ Hero（首屏）
**大标题（诗性 / 宣言感）：**
```
半梦半醒
永远年轻，永远热泪盈眶
```

**副标题（小字）：**
```
我在做社区、文化与技术之间的连接
```

**两个按钮：**
- 查看项目
- 联系我

#### 2️⃣ About Me（关于我）
- 3-5 段短句，偏叙事，不要简历腔
- 强调个人关注点和价值观
- 不是技能列表，而是故事

**内容方向：**
- 我关注人与人之间的连接
- 我相信真实关系和长期建设的力量
- 我在探索社区、文化与技术的交集

#### 3️⃣ Projects（项目展示）
**项目一：拼团社区项目**
- 简介：尝试搭建一个围绕拼团、信任与协作的社区实验
- 角色：社区发起 / 组织 / 产品与运营协调
- 内容：关注信任、协作与真实关系的建立

**项目二：风水文化项目**
- 简介：尝试用现代方式重新理解与表达传统风水与东方文化
- 角色：内容策划 / 文化转译 / 社群搭建
- 内容：让传统文化回到"生活智慧"本身

**项目三：区块链社区项目**
- 简介：围绕区块链与 Web3 的学习型社区
- 角色：社区建设 / 活动组织 / 对外合作
- 内容：重点不在投机，而在共识、协作与长期建设

**特点：**
- 强调"社区"而非"炫技术"
- 每个项目有详情页，深入讲述故事
- 项目顺序可调整

#### 4️⃣ Writing / Thoughts（一些想法）
**标题：** 一些想法 / 一些记录

**当前状态：** 正在整理中

**功能：**
- 支持未来加入文章列表
- 文章详情页
- Markdown 渲染

#### 5️⃣ Contact（联系我）
**简短真诚的一句话：**
```
如果你对社区、文化或长期建设感兴趣，欢迎联系我。
```

## 🎨 设计细节

### 排版
- **标题字体大小**：
  - H1: 3.5rem (56px) - 首屏标题
  - H2: 2.25rem (36px) - 板块标题
  - H3: 1.5rem (24px) - 子标题
  - Body: 1rem (16px) - 正文

- **行高**：1.6 - 1.8（提高可读性）
- **字间距**：正常（不要过紧）

### 间距
- **容器宽度**：1280px（桌面）
- **内边距**：
  - 移动端：16px
  - 平板：24px
  - 桌面：32px
- **板块间距**：100px - 150px

### 动效
- **淡入上浮**：0.6s，缓动函数 ease-out
- **Hover 效果**：0.3s，轻微阴影增加
- **滚动触发**：进入视口时动画

### 颜色应用
- **背景**：#F5F1ED（温暖米白）
- **文字**：#3C3C3C（深炭灰）
- **边框**：#E8E6E3（浅灰）
- **强调**：#2D5A4F（森林绿）
- **悬停**：#2D5A4F（绿色）
